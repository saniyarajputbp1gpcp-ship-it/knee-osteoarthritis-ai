About The Project

This project is an AI-powered Knee Osteoarthritis Detection and Severity Prediction System developed using Flask, TensorFlow, CNN, MobileNetV2 and Transfer Learning techniques. The system analyzes knee X-ray images and predicts osteoarthritis severity into four categories: Normal, Mild, Moderate and Severe.

The model was trained on labeled knee X-ray datasets with image preprocessing, augmentation and deep learning optimization techniques to improve prediction accuracy. The goal of this project is to demonstrate how Artificial Intelligence can assist in medical imaging analysis and support early diagnosis of osteoarthritis.

This web application provides an easy-to-use interface where users can upload X-ray images and instantly receive AI-generated prediction results with confidence scores.

Developed by SaRaTech using Flask, TensorFlow, CNN & MobileNetV2
